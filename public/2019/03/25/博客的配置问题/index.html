自定义格式

虽然现有的主题已经让我们的markdown很好了，但是免不了有一些想要自主定义的，给一些特定文字增加背景呀，让我们的博客更加炫酷。这里分享下我的一些设置，当然也是copy网上大佬的一些设置

更改自带的行内code样式
code {
  display:inline;
padding:.2em .6em .3em;
font-size:80%;
font-weight:bold;
line-height:1;
color:#fff;
text-align:center;
white-space:nowrap;
vertical-align:baseline;
border-radius:0;
background-color: #2780e3;
}
